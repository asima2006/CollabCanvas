# **App Name**: CollabCanvas

## Core Features:

- Real-time Drawing: Enable users to draw on a shared canvas with updates synchronized in real-time using WebSocket technology.
- Brush and Color Selection: Provide a user interface to select brush size and color for drawing.
- User Identification: Assign each user a unique color and display their name as they draw on the canvas, showing cursor locations to represent active drawing spots.
- Undo/Redo Functionality: Implement undo and redo features that allow users to revert or reapply the latest changes on the canvas, managed via a shared history of actions.
- Responsive Design: Ensure the canvas adapts to different screen sizes and devices, providing a consistent drawing experience across desktop and mobile platforms.
- Conflict Resolution: Implement basic conflict resolution to ensure the latest drawing action is reflected across all connected users.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), providing a professional and calming backdrop for creative collaboration.
- Background color: Very light indigo (#F0F2FA), a muted tone that ensures the drawing takes center stage without causing eye strain.
- Accent color: Soft violet (#9575CD), utilized to highlight active tools or user indicators, providing visual interest without overwhelming the user.
- Body and headline font: 'Inter', a sans-serif font known for its clean, modern appearance and excellent readability across different screen sizes.
- Use simple, outlined icons for tools like brush, eraser, and undo/redo, ensuring quick visual understanding.
- A clean, intuitive layout that places the canvas centrally with tool options arranged neatly around it for easy access.
- Subtle cursor animations and smooth transitions between tools for a fluid user experience.
